# Domain Docs

This repository uses a single domain context.

- Domain glossary: `CONTEXT.md`
- Architecture decisions: `docs/adr/`
- Implementation progress: `docs/实施进度.md`
- System design: `docs/系统设计.md`
- Agent design: `docs/Agent架构说明.md`
