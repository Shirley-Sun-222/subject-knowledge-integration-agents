- 已在图谱 metrics 和前端状态区暴露 LLM 抽取 / 关键词降级抽取状态，便于识别未配置 LLM API 时的知识点质量限制。
- 已新增本地 LLM 安全配置与连通性检查脚本：`scripts/configure_local_llm.py` 通过隐藏输入写入 `.env`，`scripts/check_llm.py` 检查 `complete_text` 和 `complete_json`，不打印 API Key。
- 已补充 README 中 DeepSeek、GitHub 发布和魔搭创空间环境变量部署说明。真实 API Key 只允许放入本地 `.env` 或部署平台环境变量，不能进入 Git 仓库。

- 已将 Docker 容器默认端口改为魔搭 Docker 创空间要求的 `7860`；本地 `docker-compose.yml` 继续映射到宿主机 `8000` 便于本地访问。
